Lossless Compression Techniques

  • Huffman Coding: The encoded bits are unique for each data and stored in Huffman code dictionary.
  • Arithmetic Coding: Uses continuous numbers that assign fractional values to the symbols based on their probabilities.
  • Lempel-Ziv-Welch (LZW) Encoding: Transforming strings into codes to get the image compressed.

Lossy Compression Techniques

  • Transform Coding: An universally used compression technique for lossy compression that is applied for digital signal processing, data compression, and image and audio compression.
  • Discrete Cosine Transform (DCT): Means a substantial amount of information is stored in an extremely low frequency component of a signal and other frequencies have very small data that can be contained by using little number of bits.
  • Discrete Wavelet Transform (DWT): Convert image pixels into wavelets, which are then used for wavelet-based compression and coding.
  • Fractal Compression: Seeks to construct an approximation of the original image that is precise enough to be acceptable, closer to the original at higher compression ratios.
