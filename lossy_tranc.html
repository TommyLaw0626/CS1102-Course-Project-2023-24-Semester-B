Transform Coding

Transform codding is an universally used compression technique for lossy compression that is applied for digital signal processing, data compression, and image and audio compression.

  • Image data is first converted into another domain so that the data is not correlated in the new domain for further processing
  • Operates on a reversible linear transform coefficients of the image
  • The choice of transform and quantization parameters depends on the application and the desired strength of compression
    • For example: DCT, DFT, Walsh etc. Steps:
    1. Transform: Convert the time-domain signal into a frequency-domain representation
    • Discrete Fourier Transform (DFT): most commonly used transform
    • Discrete Consine Transform
    • Wavelet Transform
    • Karhunen-Loeve Transform (KLT)
    2. Quantization: Quantizing the coefficients by rounding off the coefficients to a smaller number of bits
    • This process leads to less precision of the coefficients and results in information loss
    3. Encoding: The coefficients are encoded using coding techniques which are shown in the lossless image compression algorithm
    • Reducing the number of bits required to represent the coefficients
    4. Decoding: The reverse of the encoding process, which is the encoded coefficients being decoded to obtain the quantized coefficients
    • Decoded coefficients are utilized to reconstruct the frequency-domain signal
    5. Inverse Transform: A technique applied to the reconstructed signal to obtain the time-domain representation of the compressed signal
