Discrete Cosine Transform (DCT)

Discrete Cosine Transform (DCT) is a transform technique that has very strong energy compaction, which means a substantial amount of information is stored in an extremely low frequency component of a signal and other frequencies have very small data that can be contained by using little number of bits (usually 2 or 3 bit maximum)

Steps:

  1. Fetch image file information with pixel value in terms of integer within the range of 0 to 255
  2. Divide the pixel value in block of 8×8 matrix
  3. Apply Discrete Cosine Transform (DCT) on the block of data in Step 2.
  4. Observe that more than 90% of the data will be in the lower frequency component
  5. Perform a 2-dimensional Discrete Cosine Transform on the data observed in Step 4 to observe the output
