Huffman Coding

  • The encoded bits are unique for each data and stored in Huffman code dictionary.
  • The basis of Huffman code is constructed based on Huffman binary tree. The more frequently the symbols appear, the less bits of number assigned to it.
  • Widely use for JPEG and PNG.
